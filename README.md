# Remontada-Epica-De-Diseno
Este repositorio es del grupo que hara la remotada mas epica de una materia para salvar el promedio
# Tema
## Clínica: agenda de citas y atención
Propósito: gestionar agenda médica por especialidad y atender citas. Actores: Paciente, Recepción, Médico, Administrador.
Reglas esenciales
* Un médico no puede tener dos citas en el mismo horario.
* Estados de cita: solicitada, confirmada, cancelada, atendida.
* Registro de atención (motivo y diagnóstico breve).
## MVP
1. Agenda por médico/especialidad con horarios.
2. Solicitud, confirmación y cancelación de cita.
3. Recordatorios (correo/SMS simulado).
4. Registro de atención y cierre de cita.
5. Listados por estado, médico y fecha.
6. Identificación con CI.
